# 🦷 Clove Dental Landing Page

A modern, responsive, and visually engaging dental healthcare landing page developed using **React.js** and **Vite**. The project was built by converting a Figma design into a production-ready web application while maintaining design accuracy, responsiveness, accessibility, and performance.

---

## 🌐 Live Demo

🚀 **Live Application:**  
https://clove-dental-hdclrdefp-gouthamraju13s-projects.vercel.app

---

## 📖 Project Overview

The Clove Dental Landing Page is a healthcare-focused web application designed to provide users with information about dental treatments, services, clinic locations, patient testimonials, and frequently asked questions through a modern and intuitive user interface.

The project emphasizes:

- Pixel-perfect Figma implementation
- Responsive design across all devices
- Component-based architecture
- Accessibility best practices
- Interactive user experience

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Gouthamraju13/Clove-Dental.git
```

### Navigate to Project Directory

```bash
cd Clove-Dental
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
